Create rout for drone flight

This option allows you to calculate distance and amount of photos drone needs to take to cover all territory you marked. After program calculates all needed resources you will be able to see the optimal rout. If you satisfied with result you have an opportunity to download rout with coordinates or you can upload this rout to drone.

Make best focused photo

This option allows you to create well focused photo of some object or place. If for some reason photos of that object are unfocused in some places - just upload all of them and this program will create best focused photo out of those you uploaded. If you satisfied with result you have an opportunity to download result.

Map

This option is used to create maps from photos your drone took while flying on certain territories. In order to this option to work - upload all photos your drone took to this program and algorithm will create a map. If you satisfied with result you have an opportunity to download result.

Check field for watering

In order to check if certain field needs to be watered you just need to upload photos with fields and algorithm will check all fields and mark those which in need of water. If you satisfied with result you have an opportunity to download result.

Compare

After taking photos of the same object but in different period of time you want to see the difference between those photos. This option allows you to upload photos you want to compare and as a result you will have photos with marked difference. If you satisfied with result you have an opportunity to download result.
